# FocusFlow

A Google-inspired productivity web application built using HTML, CSS, and JavaScript to help users organize tasks, manage notes, track focus sessions, and improve productivity.

## 🌐 Live Demo

https://nitinstdio.github.io/FocusFlow/

## ✨ Features

- ✅ Task Management
- 📝 Notes
- ⏱️ Pomodoro Timer
- 📅 Calendar
- 📊 Productivity Dashboard
- 🌙 Dark & Light Mode
- 📱 Responsive Design

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
index.html
style.css
script.js
dashboard.html
dashboard.js
tasks.html
notes.html
pomodoro.html

```
